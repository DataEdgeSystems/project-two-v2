Using materialize color - 

for background color use the color name and the version (lighten(1-5), darken(1-4), accent(1-4)) 
fot applying color to the text use (color-text)
Different colors we have in materialize are: 
red, pink, purple, deep-purple, indigo, blue, light-blue, cyan, teal, green, light-green, lime, yello, amber, orange, deep-orange,( brown, grey, blue-grey does not have accent)
